# Оценка результатов A/B-теста - интернет-магазин BitMotion Kit

*Проект в рамках обучения (Яндекс Практикум)*

Стек: Python (pandas, numpy, matplotlib, seaborn, scipy.stats, statsmodels)

---

## О проекте

Интернет-магазин BitMotion Kit (геймифицированные товары для здорового образа жизни) разработал новую версию сайта, чтобы упростить интерфейс и увеличить конверсию. Задача - оценить корректность проведения A/B-теста и проанализировать его результаты.

**Гипотеза:** упрощение интерфейса приведет к тому, что в течение 7 дней после регистрации конверсия увеличится минимум на 3 процентных пункта.

**Параметры теста:**
- Название: interface_eu_test
- Группа A: контроль (старый интерфейс)
- Группа B: тест (новый интерфейс)

---

## Что сделано

### 1. Загрузка и предобработка данных

**Таблица участников (participants):**
- 14525 записей, 4 столбца (user_id, group, ab_test, device)
- Обнаружено 887 пользователей (6.5%), участвующих одновременно в двух тестах
- После очистки осталось 9963 пользователя

**Таблица событий (events):**
- Исходно 787286 записей
- Удалены системные события GLOBAL (58495 записей)
- Удалены технические дубликаты (38660 записей, 5.3%)
- После очистки: 690131 событие от 144183 пользователей

### 2. Оценка корректности теста

- Распределение после очистки: A = 4952 (49.7%), B = 5011 (50.3%)
- Проверка достаточности выборки: требуется 3762 пользователя в группе, есть >5000
- Горизонт анализа: первые 7 дней после регистрации
- В выборку попало 58411 событий (86.4% от всех)

### 3. Результаты A/B-теста

| Показатель | Группа A (контроль) | Группа B (тест) |
|------------|---------------------|-----------------|
| Посетители | 4952 | 5011 |
| Покупатели | 1377 | 1480 |
| Конверсия | 27.8% | 29.5% |

**Изменение конверсии:**
- Абсолютное: +1.73 процентных пункта
- Относительное: +6.2%

### 4. Статистическая проверка

**Метод:** z-тест для пропорций (односторонний)

**Результат:** p-value = 0.0283

**Уровень значимости:** α = 0.05

**Вывод:** p-value < α -> разница статистически значима

---

## Ключевые выводы

1. Новый интерфейс ДЕЙСТВИТЕЛЬНО увеличивает конверсию (статистически значимо)
2. Но целевой эффект НЕ ДОСТИГНУТ: ожидали +3 п.п., получили +1.73 п.п.
3. Выборка достаточная - результатам можно доверять
