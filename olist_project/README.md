# Olist - анализ бразильского маркетплейса

**Тестовое задание (полный анализ для 3 отделов)**  
Стек: Python (pandas, seaborn, plotly), SQL (PostgreSQL), Yandex DataLens

---

## О проекте

Olist - бразильский маркетплейс, связывающий продавцов, покупателей и логистику.  
Задача: подготовить аналитику и рекомендации для трех отделов:

- **Логистика** - оценка сроков доставки, выявление проблемных регионов, влияние задержек на удовлетворенность
- **Продажи** - анализ динамики выручки, региональной структуры, среднего чека
- **Качество** - оценка работы продавцов, сегментация, выявление проблемных зон

**Формат выполнения:** полный анализ (SQL-витрины + Python-анализ + дашборд + аналитическая записка)

---

## Что сделано

### 1. SQL-витрины (PostgreSQL)
Созданы 3 витрины данных:
- `logistics_vitrina` - по заказам: сроки доставки, просрочка, регион, оценка клиента
- `sales_vitrina` - по штатам и месяцам: выручка, число заказов, средний чек
- `quality_vitrina` - по продавцам: средняя оценка, доля негативных отзывов, регион


### 2. Python-анализ (Jupyter Notebook)
- Загрузка и предобработка данных
- Логистика: динамика просрочек, карта сроков по регионам, корреляция с оценками
- Продажи: динамика выручки, топ-регионы, темпы роста, средний чек
- Качество: распределение оценок продавцов, выявление проблемных селлеров

### 3. Дашборд (Yandex DataLens)
Интерактивный дашборд с тремя блоками:
- Логистика (просрочки по месяцам, карта сроков доставки)
- Продажи (динамика выручки, топ-регионы)
- Качество (рейтинги продавцов, связь с задержками)

Общие фильтры: дата, регион.

### 4. Аналитическая записка
Подробные выводы и рекомендации для каждого отдела (см. файл `analytical_report.md`)

---

## Ключевые выводы

### Логистика
- Общая доля просрочек - 7.5%, с пиками до 21% в отдельные месяцы
- Регионы-антилидеры: AL (24%), MA (19.6%), PI, CE, SE, RJ
- Корреляция между просрочками и оценками клиентов: -0.708 (сильная обратная)

### Продажи
- Топ-5 регионов дают 73.2% выручки, штат SP - 37.4% (зона высокого риска)
- SP вырос на 172% (абсолютный лидер), регионы-драйверы: RR (+451%), RN (+179%)
- Малые регионы (PB, AC, AP, AL) имеют максимальный средний чек (до 267)

### Качество
- 85% активных продавцов работают "хорошо" и выше
- Проблемные продавцы - 2.2%, но 60% из них в штате SP
- Корреляция между опытом и оценкой: -0.253 (слабый отрицательный)

### Главный риск
Гиперконцентрация бизнеса в SP: 37% выручки + 60% проблемных продавцов
