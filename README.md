| Инструмент | Навык |

| **Python** | pandas, numpy, seaborn, plotly, scipy.stats (EDA, очистка, визуализация, статистика) |
| **SQL** | JOIN, оконные функции, подзапросы, агрегации, оптимизация запросов |

- Юнит-экономика, когортный анализ, воронки
