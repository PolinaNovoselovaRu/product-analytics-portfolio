## Описание дашборда

Дашборд состоит из двух вкладок:

### Вкладка 1. Аналитика рынка недвижимости

- Фильтр по типам населенных пунктов
- Показатели: количество квартир, средняя цена, средняя площадь, среднее количество дней продажи
- Топ-5 населенных пунктов по скорости продажи
- Графики с возможностью детализации по дням/неделям/месяцам/годам

### Вкладка 2. Детализация рынка

- Таблица населенных пунктов с сортировкой и градиентной раскраской по стоимости
- Столбчатые диаграммы: средняя цена за кв.м по планировке, типу жилья, наличию парка/водоема
- Активность публикаций по дням недели (зеленый - самый активный день)
- Линейный график сезонности по годам

---

## Как использовать

1. Перейдите по ссылке
2. Используйте фильтры для детализации данных
3. На вкладке "Аналитика рынка недвижимости" переключайте гранулярность (день/неделя/месяц/год)
4. На вкладке "Детализация рынка" анализируйте факторы, влияющие на стоимость
